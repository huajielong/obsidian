# OpenClaw 信息库汇总

## 核心主旨

OpenClaw 是一个 AI Agent 平台/工具，具备 Skills 插件生态、Session 管理、多模型支持等能力。本摘要汇总了其功能更新、变现模式、插件生态和常见问题。

## 产品定位

OpenClaw 是一个开源 AI Agent 平台，支持 GPT-5、Claude 等多种模型，具备插件化 Skills 系统、Context 引擎、会话管理等能力。GitHub 地址：https://github.com/openclaw/openclaw

## Skills 插件生态

截至 2026 年 3 月，总 Skills 数量 1700+，分类 31 个，社区评分 4.5+ 的超过 200 个。

**必装 Top Skills：**
- skill-vetter（安全扫描）
- tavily-search（实时搜索）
- agent-browser（浏览器自动化）
- summarize（长文本摘要）
- self-improving-agent（自我进化）
- find-skills（技能导购）

**安装方式**：ClawHub CLI / openclaw CLI / 手动 / GitHub

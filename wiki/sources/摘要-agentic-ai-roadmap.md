# Agentic AI 学习路线

## 核心主旨

从零开始进入 Agentic AI 领域的系统学习路线，包含 Stage0 基础准备（Python、Git、CLI、REST API、YAML/JSON/jq）到 Stage1 LLM 基础的结构化路径。

## Stage0 基础准备

### Python
资源推荐：Python Crash Course、Real Python、Corey Schafer YouTube、Boot.dev、runoob.com

**并发模型实战**：
- async/await（单线程并发 IO）：适用于高并发 HTTP 请求、爬虫、网关
- 多线程（混合任务/阻塞 IO 库）：适用于文件读写、SDK 调用
- 多进程（CPU 密集）：适用于图像处理、数值计算

### Git
资源推荐：Pro Git book、Atlassian Git Tutorials、Oh Shit, Git!?!、git-flight-rules

### CLI / Shell
资源推荐：The Art of Command Line、Learn Shell、explainshell.com

### REST API
- **REST API**：接口设计风格/架构规范
- **FastAPI**：Python 生态中实现 REST API 的现代 Web 框架
- 主流语言框架横评：Python（FastAPI/Flask/DRF）、Java（Spring Boot）、Node.js（Express/NestJS）、Go（Gin）

### YAML / JSON / jq
资源配置和查询工具基础，jq 在 Agent 工作中常用。

## 核心摘要

该资料是 "awesome-agentic-ai-zh" 系统学习路线图的 **Stage 0——基础准备（Foundations）**，列出了学习 AI Agent 开发前必须掌握的五项先修技能：Python 编程（函数、类、async/await）、Git 版本控制（clone/commit/push/merge）、命令行操作（cd/ls/mkdir）、REST API 调用（GET/POST/JSON/auth header）、YAML/JSON 配置读写。每个技能配有极简练习，并推荐精选学习资源。核心观点：**不要跳过这些基础——大多数 AI Agent 教学都预设你已经会了，没准备好会在奇怪的地方卡关。**

## 关键提炼

- **五项先修技能**：Python → Git → CLI → REST API → YAML/JSON
- **时间估算**：1-2 周（约 5-15 小时），已具备可跳过
- **核心理念**：在这一周投资基础，可省下后面 10 周以上的挫折
- **GitHub API 练习亮点**：用 Personal Access Token 体验 401 vs 200 的差异，理解 API auth 机制——这是 production agent 一定会用到的技能
- **资源推荐**：Python Crash Course（书）、Boot.dev（互动）、Pro Git（免费参考书）、Oh Shit, Git!?!（急救手册）、The Art of Command Line（CLI 大全）、HTTPie（替代 curl）、jq（命令行 JSON 处理）

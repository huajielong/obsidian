## 核心摘要

System Prompt 控制力实验：使用 deepseek-v4-pro，固定 User Message（"请帮我解释什么是租赁合约"），仅改变 System Prompt 为三种人格（冷酷技术专家、热情营销写手、三行格式机器人），对比输出差异。关键发现：System Prompt 不只控制"怎么说"，也实质性地改变"说什么"——风格越强，内容深度往往越浅；格式约束比语气约束更稳定。

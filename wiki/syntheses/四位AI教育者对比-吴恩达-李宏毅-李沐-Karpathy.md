## 一、人物简介

### 🧑‍🏫 吴恩达（Andrew Ng）— "AI 启蒙之父"

- **身份**：斯坦福客座教授、Coursera 联合创始人、DeepLearning.AI 创始人
- **经历**：Google Brain 创始成员、百度前首席科学家
- **教学风格**：系统化、由浅入深、数学推导友好
- **最适合**：**零基础入门者**、想建立"AI 大局观"的人
- 查看实体：[[Andrew_Ng]]

### 🧑‍🏫 李宏毅（Hung-yi Lee）— "中文 AI 教学天花板"

- **身份**：台湾大学电机工程学系副教授
- **经历**：台大终身研究荣誉，专注语音/NLP/生成式 AI 研究
- **教学风格**：生动案例（宝可梦）→ 硬核原理 → 每年紧跟 SOTA
- **最适合**：**中文母语学习者**、追求课程时效性的人
- 查看实体：[[Hung_yi_Lee]]

### 🧑‍🏫 李沐（Mu Li）— "动手派实战导师"

- **身份**：亚马逊 AWS 首席科学家
- **经历**：加州大学伯克利博士、亚马逊 AI / 百度 IDL 主任
- **教学风格**：代码驱动、教科书同步、论文精读
- **最适合**：**想一边学一边写代码的人**、追求工程落地能力的人
- 查看实体：[[Mu_Li]]

### 🧑‍🏫 Andrej Karpathy — "从零构建派硬核教学"

- **身份**：OpenAI 创始成员、前 Tesla AI 总监
- **经历**：斯坦福博士（师从 Fei-Fei Li）
- **教学风格**：从零手写所有代码 → 极简 → 深刻
- **最适合**：**想彻底搞懂底层原理的人**、有一定编程基础的工程师
- 查看实体：[[Andrej_Karpathy]]

### 🧑‍🏫 李飞飞（Fei-Fei Li）— "计算机视觉教母"

- **身份**：斯坦福大学终身教授、斯坦福 HAI 联合主任、美国工程院院士
- **经历**：ImageNet 发起者、前斯坦福 AI 实验室主任、World Labs CEO
- **教学风格**：前沿理论 → 系统体系 → 科研落地
- **最适合**：**想系统学习计算机视觉的人**、对 CV 研究感兴趣的人
- 查看实体：[[Fei-Fei_Li]]

### 各层详解

#### 🟢 第 1 层：AI 入门直觉 — 吴恩达·机器学习

| 课程 | 说明 | 前置要求 | 预计时长 |
|------|------|---------|---------|
| **ML Course** (Coursera) | AI 史上最友好的入门课，建立 ML 基本概念：监督/无监督学习、线性回归、逻辑回归、神经网络直觉 | 无（高中数学 + 基础编程） | 2-3 个月（每周 3-5h） |

> **关键产出**：理解"机器学习能做什么"，建立术语体系和直觉。**这是唯一一个零基础可以直接开始的课程。**

---

#### 🔵 第 2 层：深度学习理论体系 — 二选一路线

完成第 1 层后，你应该系统学习深度学习理论。此时有两条平行路线：

| 对比项 | 路线 A：吴恩达 DL 专项 | 路线 B：李宏毅 ML 课程 |
|--------|----------------------|----------------------|
| **课程** | DL Specialization（5 门课） | 台大 ML 2025/2026 |
| **语言** | 英文（中文字幕） | **中文（普通话）** |
| **内容定位** | 经典 DL 理论：NN/CNN/RNN/超参调优/结构化 ML 项目 | **紧跟 SOTA**：含生成式 AI / AI Agent / LLM 等最新内容 |
| **理论深度** | ⭐⭐⭐ 偏上层、重应用 | ⭐⭐⭐⭐ 公式推导更多 |
| **学习曲线** | 🌱 平缓 | 🌿 中等 |
| **预计时长** | 3-4 个月 | 2-3 个月 |
| **推荐场景** | **零基础程序员** 首选 | **中文母语者**、追求 **前沿性** 首选 |

> **💡 最佳组合**：先路线 A 建立系统框架（3 个月），再用路线 B 补前沿视角（选听最新章节 3-4 讲即可）。这样既有经典深度，又有时效性。

---

#### 🟡 第 3 层：动手能力 + 原理深挖（可并行）

第 2 层之后，你有三条可**并行**或**按需选取**的路线：

##### 第 3A 层：代码实战 — 李沐·动手学深度学习 v2

| 课程 | 说明 | 前置要求 | 预计时长 |
|------|------|---------|---------|
| **D2L v2** (~142 集) | 每个概念都有代码实现，教科书级别的系统性 | 第 2 层理论（理解 NN/CNN/RNN） | 2-3 个月（建议每天 1-2 集 + 跑代码） |

> **这个阶段是"从听懂到写出来"的质变**。李沐带你手写每一行代码，把第 2 层学到的理论知识转化成工程能力。不是看一遍就过，而是**边看边敲**。

##### 第 3B 层：底层原理深挖 — Karpathy·Zero to Hero（第 1-6 讲）

| 讲次 | 核心内容 | 与第 2 层（李沐）的关系 |
|------|---------|---------------------|
| 第 1 讲 | micrograd：手写 autograd | 从 Python 层面理解 DL 框架的数学基础 |
| 第 2 讲 | makemore：字符级语言模型 | 理解"语言模型"最朴素的形式 |
| 第 3 讲 | MLP + 过拟合 + 学习率 | 与李沐 MLP 章节**互为印证**，一个从代码走通，一个从数学走通 |
| 第 4 讲 | 激活函数 + BatchNorm | 李沐讲"怎么用"，Karpathy 讲"为什么这样" |
| 第 5 讲 | Backprop ninja：手算梯度 | 李沐用 autograd 自动求导，Karpathy 教你**手算验证** |
| 第 6 讲 | WaveNet：MLP → CNN | 从 MLP 自然过渡到卷积架构 |

> **💡 与李沐 D2L 的协同策略**：
> - 学完李沐的"多层感知机"章节后 → 看 Karpathy 第 3-4 讲
> - 学完李沐的"反向传播"章节后 → 看 Karpathy 第 5 讲
> - 学完李沐的"CNN"章节后 → 看 Karpathy 第 6 讲
>
> 这种 **"李沐教你怎么做 → Karpathy 告诉你为什么"** 的配对，是最高效的学习方式。

##### 第 3C 层：前沿窗口 — 李宏毅最新课程

| 内容 | 说明 |
|------|------|
| 大模型原理 / AI Agent / RAG / 生成式 AI | 每年更新，保持你对"AI 正在发生什么"的感知 |
| **推荐时机** | 在第 2 层之后，**贯穿整个学习过程**，碎片时间听 |

> **不是系统性学习**，而是保持与前沿的触觉——每周看 1-2 个切片。

### 核心原则：不要线性看完再下一个

很多人会陷入"看完一个老师的全部课程，再开始下一个"的误区。**最高效的方式是交叉并行**：

```
┌────────────────────────────────────────────────────────┐
│  周次 │ 主线任务            │ 辅助任务                    │
│──────┼────────────────────┼───────────────────────────│
│ 1-8  │ 吴恩达 ML Course   │ —                          │
│ 9-20 │ 吴恩达 DL 专项      │ 李宏毅前沿切片（碎片时间）    │
│ 21-28│ 李沐 D2L v2        │ 对应当周主题的 Karpathy 讲次 │
│ 29-32│ CS231n / GPT 系列   │ 李沐论文精读（主题对应）      │
└────────────────────────────────────────────────────────┘
```

> **核心策略**：同一主题在**理论（吴恩达/李宏毅）→ 代码（李沐）→ 原理深挖（Karpathy）** 三个层次上穿透，而不是每个老师按从头到尾"线性推进"。

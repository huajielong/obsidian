# Codex

## 定义

Codex 是 OpenAI 推出的 AI 编码智能体，是 [[Harness_Engineering]] 实验中的核心执行引擎。在 OpenAI 的实验中，Codex（基于 GPT-5）在 5 个月内自主编写了约 100 万行代码。

## 关键信息

- **开发商**：OpenAI
- **核心能力**：自主编程、测试、调试、部署
- **在 Harness Engineering 中的角色**：作为 Agent 层的执行主体，在人类设定的约束和反馈回路中工作
- **访问方式**：通过 Codex CLI 使用

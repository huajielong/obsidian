# Telegram

## 定义

Telegram 是一款跨平台即时通讯软件。在 AI Agent 生态中，它常作为 Agent 与用户之间的聊天界面接口，用户可通过 Telegram bot 与部署在云端的 Agent 交互。

## 关键信息

- **用途**：Agent 聊天界面接口
- **相关模式**：Cloud-deployed Agent 常用 Telegram bot 作为交互前端
- **代表项目**：[[Hermes_Agent]] 等云端 Agent 支持通过 Telegram 交互

## 关联连接

- [[AgentParadigms]] — 云端部署范式中 Telegram 作为 Agent 接口
- [[Hermes_Agent]] — 支持 Telegram bot 交互的开源 Agent
- [[Slack]] — 同为 Agent 聊天界面接口

# Goose

> ★ 43k+ · Apache-2.0 · [GitHub](https://github.com/block/goose)（原 block/goose，现指向 `aaif-goose/goose`）

**开源、可扩展的 AI agent**，超出纯 code suggestion——能 **install / execute / edit / test**，搭配任何 LLM。

## 核心特性

- **多 LLM 支持**: 支持多家 LLM provider
- **MCP 集成**: 支持 Model Context Protocol 扩展
- **三接口**: 同时提供 desktop app、CLI、API 三种交互方式
- **全流程**: 不仅能写代码，还能安装依赖、执行、测试

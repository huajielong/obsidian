# Cursor

Cursor 是编辑器集成的 AI 结对编程工具。在 AI 编辑器类工具中采用度最高，常作为比较其他 IDE agent 的基准。

## 核心定位

- **产品形态**: 基于 VS Code 的 AI 原生代码编辑器
- **核心能力**: 自动补全 + 改写 + 解释，编辑器内 AI 对话
- **位置**: Tier 0 工具（适合所有开发者，学习成本为 0）
- **对标**: Copilot（GitHub）、其他 IDE agent

## 在开发者工作流中的角色

- 适合 **AI 结对编程** 场景：写到半忘 syntax / 想不到 method 名 → 自动补全
- 适合 **多文件重构** 入口级工具：轻量级 refactor 场景
- 常作为 **Tier 0 → Tier 1** 的起点：Cursor/Copilot → Claude Code

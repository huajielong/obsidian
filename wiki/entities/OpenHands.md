# OpenHands

> ★ 72k+ · MIT · [GitHub](https://github.com/All-Hands-AI/OpenHands)（前身为 **OpenDevin**）

**开源自主软件开发 agent**。设计上比 [[Aider]] 和 [[Claude_Code]] 更激进——agent 自己跑 sandbox、自己 commit，适合 **"整个 issue 丢给它解"** 场景。

## 核心定位

- **自主性**: 全自主模式，从理解 issue 到提交代码全程 agent 驱动
- **Sandbox 运行**: 在安全沙箱中执行代码
- **典型场景**: 完整的 issue 解决，而非逐行编码辅助
- **对比**: 比 Aider / Claude Code 的 human-in-the-loop 更激进

# Roo Code

> ★ 23k+ · Apache-2.0 · [GitHub](https://github.com/RooCodeInc/Roo-Code)

VS Code 的 coding agent，采用 **"多种专业模式"** 的设计，与 [[Cline]] 的单一 agent flow 不同。

## 核心定位

- **产品形态**: VS Code extension
- **核心差异**: 多专业模式（multi-mode）vs Cline 的单 agent flow
- **定位**: VS Code 用户想 multi-mode 替代方案的选择
- **对标**: Cline（同生态直接对手）

## 对比 Cline

| 维度 | Roo Code | Cline |
|---|---|---|
| 模式设计 | 多种专业模式 | 单一 agent flow |
| ★ | 23k+ | 61k+ |
| 定位 | multi-mode 替代方案 | autonomous in-IDE agent |

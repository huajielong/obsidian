# Hermes Agent

## 定义

Hermes Agent 由 Nous Research 开发的开源自主 CLI Agent（★ 213k+），支持云端部署和 CLI 模式运行。在 [[AgentParadigms]] 的五种部署范式中，Hermes Agent 是 Cloud-deployed（云端部署）和 BYO-LLM CLI 两种范式的代表项目之一。

## 关键信息

- **开发商**：Nous Research
- **开源许可**：开源（★ 213k+）
- **部署模式**：云端部署（VPS/服务器）、CLI 模式
- **接口方式**：Telegram / Slack 消息界面、终端 REPL
- **路由能力**：200+ provider 路由

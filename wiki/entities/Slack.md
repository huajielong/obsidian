# Slack

## 定义

Slack 是一款企业协作通讯平台。在 AI Agent 生态中，它常作为 Agent 与用户之间的聊天界面接口，用户可通过 Slack bot 与部署在云端的 Agent 交互。

## 关键信息

- **用途**：Agent 聊天界面接口
- **相关模式**：Cloud-deployed Agent 常用 Slack bot 作为交互前端
- **代表项目**：[[Hermes_Agent]] 等云端 Agent 支持通过 Slack 交互

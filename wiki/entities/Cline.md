# Cline

> ★ 61k+ · Apache-2.0 · [GitHub](https://github.com/cline/cline)

**VS Code extension**，提供 autonomous in-IDE agent 体验。支持 tool use、浏览器操作、step-by-step approval 机制。

## 核心定位

- **产品形态**: VS Code 插件（非独立 CLI）
- **核心能力**: 自主 agent 在 IDE 内完成编码任务
- **审批模式**: step-by-step approval，human-in-the-loop
- **对比**: 与 [[Roo_Code]] 同为 VS Code 生态，但 Cline 采用单一 agent flow

## 适用场景

- **VS Code 用户** 想获得 IDE-native agentic dev 体验的好选项
- **Code review**: 能审查自己的 diff
- **Tier 1** 工具：适合从 IDE 起步进阶 CLI agent 的用户

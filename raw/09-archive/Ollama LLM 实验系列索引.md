## 实践建议

1. **生产环境使用 `temperature=0`**：保证输出一致性
2. **需要多样性时用 `temperature=0.7`**
3. **云端 API 比本地快 2~14 倍**，成本极低（百次 ~¥0.006~0.03）
4. **Fine-tuning 决定模型"性格"**：0.8B 模型回复可以是 3B 模型的 6.5 倍长

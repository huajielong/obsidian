# 实验五：跨平台成本对比（本地 vs 云端）

> [[Ollama LLM 实验系列索引]]

## 实验方法

对同一 Prompt（"你好"），在 4 个模型上各重复 100 次请求（`temperature=0.0`），统计延迟、Token 消耗和成本。

## 实验结果

| 平台 | 模型 | 总耗时 | 平均延迟 | 输入 T/次 | 输出 T/次 | 费用(100次) | 生成速度 |
|:---|:---|:---:|:---:|:---:|:---:|:---:|:---:|
| **Ollama** | llama3.2:3b (3B) | 230s | 2.30s | 27 | 18 | **¥0** | 7.8 tok/s |
| **Ollama** | qwen3.5:0.8b (0.8B) | 1,463s | 14.63s | 11 | 278 | **¥0** | 19.0 tok/s |
| **DeepSeek** | V4 Flash (云端) | 265s | 2.65s | 5 | 144 | **¥0.029** | ~54 tok/s |
| **豆包** | Seed Character (云端) | **45s** | **0.45s** | 31 | 15 | **¥0.006** | ~33 tok/s |

> DeepSeek 为非高峰时段价格（9:00~12:00 / 14:00~18:00 高峰×2）；豆包为参考价。

### 每百万次调用成本换算

| 平台 | 每百万次费用（"你好"级短请求） |
|:---|:---:|
| Ollama (本地) | ¥0 + 电费 + 硬件折旧 |
| DeepSeek V4 Flash | **~¥290** |
| 豆包 Seed Character | **~¥58** |

## 分析

**云端 vs 本地的本质取舍**：
- 云端 **快而便宜**（100 次仅 45~265 秒、几分钱），适合开发调试和低延迟场景
- 本地 **零直接成本，但需硬件投入**，适合隐私敏感、高频离线、或批量推理场景

**豆包意外胜出**：0.45 秒/次是本地模型的 5 倍快，且 100 次只要 0.6 分钱。

**DeepSeek 对简短短语回复特别长**：同样"你好"，DeepSeek 平均输出 144 Tokens（详尽自我介绍），而豆包仅 15 Tokens（简洁问候），导致 DeepSeek 的实际成本反而更高。

## 附：实验代码

```python
# 跨平台成本测试 — 核心部分
from openai import OpenAI
import urllib.request, json, time, statistics

PROMPT = "你好"
N = 100

# === Ollama 本地 ===
def test_ollama(model):
    lat, it, ot = [], [], []
    for i in range(N):
        p = {"model": model, "messages": [{"role":"user","content":PROMPT}],
             "stream": False, "options": {"temperature": 0.0}, "keep_alive": "10m"}
        t0 = time.time()
        req = urllib.request.Request("http://localhost:11434/api/chat",
            data=json.dumps(p).encode(), headers={"Content-Type":"application/json"})
        with urllib.request.urlopen(req, timeout=180) as r:
            resp = json.loads(r.read())
        lat.append(time.time() - t0)
        it.append(resp["prompt_eval_count"])
        ot.append(resp["eval_count"])
    return lat, it, ot

# === DeepSeek (云端) ===
def test_deepseek():
    client = OpenAI(base_url="https://api.deepseek.com",
                    api_key="<YOUR_API_KEY>")
    lat, it, ot = [], [], []
    for i in range(N):
        t0 = time.time()
        resp = client.chat.completions.create(
            model="deepseek-chat",
            messages=[{"role":"user","content":PROMPT}],
            temperature=0.0)
        lat.append(time.time() - t0)
        it.append(resp.usage.prompt_tokens)
        ot.append(resp.usage.completion_tokens)
    return lat, it, ot

# === 豆包 (云端) ===
def test_doubao():
    client = OpenAI(base_url="https://ark.cn-beijing.volces.com/api/v3",
                    api_key="<YOUR_API_KEY>")
    lat, it, ot = [], [], []
    for i in range(N):
        t0 = time.time()
        resp = client.chat.completions.create(
            model="doubao-seed-character-260628",
            messages=[{"role":"user","content":PROMPT}],
            temperature=0.0)
        lat.append(time.time() - t0)
        it.append(resp.usage.prompt_tokens)
        ot.append(resp.usage.completion_tokens)
    return lat, it, ot

# === 输出汇总 ===
for name, lat, it, ot in [
    ("llama3.2:3b", *test_ollama("llama3.2:3b")),
    ("qwen3.5:0.8b", *test_ollama("qwen3.5:0.8b")),
    ("DeepSeek", *test_deepseek()),
    ("豆包", *test_doubao()),
]:
    print(f"{name:<20} {sum(lat):>8.1f}s {statistics.mean(lat):>7.3f}s "
          f"{statistics.mean(it):>5.0f} {statistics.mean(ot):>5.0f} "
          f"费用: 按量计费")
```

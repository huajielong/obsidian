# 实验三：温度 = 0 确定性验证

> [[Ollama LLM 实验系列索引]] | 模型：llama3.2:3b

## 实验方法

同一 Prompt（"背诵一首唐诗的五言绝句。"），`temperature=0.0`，重复请求 20 次，检查每次的 Token 数和输出内容是否一致。

## 实验结果

```
✅ 20/20 次输出完全一致
基准: out=144 Tokens
内容: "月下独酌，人不见，独酌无味。..."
```

## 结论

`temperature=0` 时，模型的行为是 **完全确定的**。同一输入、同一权重、同一随机种子，输出 100% 一致。

> 注：某些框架（如 HuggingFace Transformers）的 `do_sample=False` + `temperature=0` 组合可能因浮点运算在不同硬件上产生细微差异，但 Ollama 的实现是稳定的。

**实践意义**：生产环境中需要确定性输出（JSON、代码、摘要）时，务必使用 `temperature=0`。

## 附：实验代码

```python
# 温度=0 确定性验证 — 取自 ollama_experiment_v3.py
prompt = "背诵一首唐诗的五言绝句。"
ref = call_ollama(prompt, temperature=0.0)
ref_text = ref["message"]["content"]
ref_out = ref["eval_count"]
print(f"  [基准] out={ref_out} | {ref_text[:60]}")

same = 0
for i in range(19):
    r = call_ollama(prompt, temperature=0.0)
    if r["message"]["content"] == ref_text and r["eval_count"] == ref_out:
        same += 1
print(f"  ✅ {same+1}/20 次完全一致 -> 温度=0 确定输出")
```

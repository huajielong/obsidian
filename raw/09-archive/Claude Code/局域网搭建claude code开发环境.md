# 局域网搭建claude code开发环境



1. 安装局域网模型服务（GPU采购+模型选型+模型部署+模型服务控制(LiteLLM)）
2. 安装node.js
3. 安装claude code
4. 常用skill集成
5. 安装playwright命令  
  
在线安装：

     安装浏览器引擎：npx playwright install chromium

     安装cli命令：npm install -g @playwright/cli@latest

     安装cli操作技能：playwright-cli install --skills

# Skill 打包提示词

把磨合好的对话，一键打包成 Skill

我们刚才的对话已经磨合出了完整的工作流程和输出标准。请现在将这个过程整理成一个标准的 Agent Skill，要求如下：

1. 创建完整的 Skill 文件夹结构
2. SKILL.md 写清楚：Skill 职责、触发场景、执行步骤、输出标准
3. references 放入我们确认过的所有格式要求和内容标准
4. 可自动化的步骤写入 scripts
5. assets 放入需要复用的模板文件
